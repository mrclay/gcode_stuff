WordPress accordion shortcode docs (alpha!)

Creating an expandable section : the [accordion] shortcode

If you prefer for some page content to appear collapsed when the page loads and for the user to be able to expand it, there's now a shortcode for this.

  1. Create your content and add a heading at the top of it which you want to use as the section title (appearing to the left of the [show] button).
  2. Click the "HTML" tab and find this content.
  3. Create a couple empty lines before/after the content.
  4. Insert [accordion] on an empty line before the header.
  5. Insert [/accordion] on an empty line below the end of the content.
  6. Preview the page/post to see the change, then update it.

You may also specify the title of the section in the shortcode: [accordion title="Read more..."]

Example usage

...existing content

[accordion]
<h3>My header</h3>
<p>my content</p>
[/accordion]

existing content...

Optional Parameters

  • expand=1 : show the accordion expanded when the page loads.
  • title="My New Title" : change the heading displayed next to the [hide/show] button.

Inserting a page in another page : the [page_accordion] shortcode

CAVEAT: Keep in mind that this shortcode has a server performance cost. If the content is only meant for this page, consider using the [accordion] shortcode.

Sometimes you might find it useful to include content from other pages/posts in another page. This can improve the user experience when folks visiting the site might want to see additional page content in the flow of reading a different page. This can be done using something called a "shortcode."

In order to embed a page within another page:

  1. Create all the necessary pages first (even if they're just placeholders)
  2. Edit the page that will include content from other pages (we'll call this the "destination page") 
  3. Click the "HTML" tab on the destination page
  4. Scroll to where you want to include the content and create 3 empty lines
  5. On the middle line enter the following shortcode:

    [page_accordion url="URL"]
  6. Replace URL with the address of a page or post within your site.
  7. Click to update the destination page.

CAVEAT: You may not be able to include home pages or other specialized pages.

Example usage

...content before.</p>

[page_accordion url="http://example.com/2011/02/03/my-post/"]

<p>Content after...

Optional Parameters

  • expand=1 : show the accordion expanded when the page loads.
  • title="My New Title" : change the heading displayed next to the [hide/show] button.
